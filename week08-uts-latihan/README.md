# Minggu 8 — UTS

> **Catatan penting:** folder ini berisi **latihan mandiri** untuk persiapan,
> menggabungkan konsep Minggu 1–7. Soal UTS yang sesungguhnya ditentukan dan
> diberikan langsung oleh dosen pengampu — bukan file ini.

## Tujuan
Mengukur pemahaman gabungan Minggu 1–7 sebelum UTS: paradigma OOP, class,
object, use case, activity, class diagram, dan sequence diagram.

## Cara Menjalankan
```bash
node index.js
```

## Soal Latihan Mandiri
1. Gambar Class Diagram lengkap dari kode di `index.js`, termasuk multiplisitas.
2. Buat Sequence Diagram untuk skenario "Budi gagal meminjam karena tunggakan".
3. Tambahkan method `kembalikanBuku(isbn)`.
4. Jelaskan (tulisan singkat) mengapa atribut dibuat `private` (`#`), dan
   risikonya bila dibuat `public`.

Sesuai kebijakan RPS: pengerjaan UTS resmi dilakukan **tanpa bantuan AI**,
kecuali dinyatakan lain oleh dosen. Latihan mandiri ini boleh dikerjakan
sambil berdiskusi, tapi biasakan diri mengerjakan tanpa AI agar siap saat UTS.
