# Minggu 15 — Presentasi Hasil Perancangan dan Peer Review

**Terkait CPMK:** 31.7

## Tujuan

Mengintegrasikan model UML, kode, dokumentasi, presentasi, peer review, dan
transparansi penggunaan AI.

## Berkas

- `demo.js` — demo end-to-end.
- `template-presentasi.md` — alur presentasi.
- `formulir-peer-review.md` — lembar umpan balik kelompok.
- `rubrik-presentasi.md` — indikator penilaian.

## Cara Menjalankan

```bash
node demo.js
```

## Tugas

1. Presentasikan masalah, requirement, diagram, kode, deployment, dan keamanan.
2. Tunjukkan minimal satu koreksi terhadap output AI.
3. Kelompok lain mengisi formulir peer review.
4. Kelompok presenter menanggapi temuan dan mencatat revisi final.
