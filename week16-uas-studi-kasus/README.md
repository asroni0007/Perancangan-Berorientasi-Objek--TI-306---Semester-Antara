# Minggu 16 — UAS

> **Catatan penting:** folder ini berisi **latihan mandiri** untuk persiapan,
> menggabungkan konsep Minggu 9–15. Soal UAS yang sesungguhnya ditentukan dan
> diberikan langsung oleh dosen pengampu — bukan file ini.

## Tujuan
Mengukur pemahaman gabungan Minggu 9–15 sebelum UAS: communication style,
state machine, prinsip OOP, design pattern, dan integrasi end-to-end.

## Cara Menjalankan
```bash
node index.js
```

## Soal Latihan Mandiri
1. Tambahkan state `DIPERPANJANG` pada tabel transisi `Peminjaman`, buktikan
   transisi ilegal ke state itu tetap ditolak oleh kode.
2. Tambahkan Observer yang mencatat setiap perubahan status `Peminjaman` ke
   sebuah *audit trail* (array log).
3. Jelaskan (tulisan) risiko pola Singleton pada `KatalogPerpustakaan` jika
   sistem di-deploy dengan banyak instance server (horizontal scaling) —
   hubungkan dengan Deployment Diagram Minggu 11.
4. Susun ringkasan proyek akhir 1 halaman, sertakan transparansi penggunaan
   AI sesuai Kebijakan Etika AI pada RPS.

Sesuai kebijakan RPS: pengerjaan UAS resmi dilakukan **tanpa bantuan AI**,
kecuali dinyatakan lain oleh dosen.
